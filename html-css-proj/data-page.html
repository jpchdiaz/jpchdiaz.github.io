About this CSV

Instead of posting the large set of CDC Data which is available for perusal on their site,
I've decided to post the FIPS-to-State conversion CSV we took from one of our TA's projects.
Combined with cleaning a dataset that uses FIPS codes, it turns the numbers in the source dataset into their proper State Abbreviations.
Converted from CSV to HTML using this converter.
